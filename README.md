## Описание проекта

CRUD-приложение позволяет управлять списком пользователей через веб-интерфейс. Реализованы следующие функции:
- Создание нового пользователя.
- Получение списка всех пользователей.
- Получение информации о конкретном пользователе.
- Обновление данных пользователя.
- Удаление пользователя.

Проект состоит из двух частей:
- **Фронтенд**: React-приложение для взаимодействия с пользователем.
- **Бэкенд**: Node.js сервер с REST API для работы с базой данных MongoDB.

---

## Технологии

- **Frontend**:
    - React
    - Axios (для запросов к API)
    - React Router (для маршрутизации)
- **Backend**:
    - Node.js
    - Express
    - MongoDB (база данных)
- **Дополнительно**:
    - Docker (для контейнеризации)
    - Docker Compose (для запуска приложения)
